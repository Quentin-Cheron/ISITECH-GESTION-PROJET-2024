# Structure de coût de projet


### Salire et prestation :

- Réalisation (~75% première année)
- Salaire internes (+65% charges)
- Prestation SSII (markup ~60%)
- Variable compétences, expérience, modalités

## Mode de Facuration

### Régie

- Facturation mensuelle
- Négociation: durée d'engagement
- 30-60 jours de paiement

### Forfait

- Paiement échelonné
- 10-30% jous de paiement
- 20-30% à la recette

## Infrastructures & Exploitation

### Hébergement

- Solutions virtualisées
- Cloud computing
- Tarification selon l'usage

### Maintenance

- Gestion utilisateur
- Supervision systèmes
- Sauvegarde/restauration

## Support & Services

### Support utilisateurs

- Niveau 1 = incident non important, 2 = incident moyen, 3 = incident urgent
- SLA définis
- Facturation selon le volume

### Coûts Additionnels

- Formation

# Chapitre 2.6 La réglemtation concernant la protection des données personelles

## Protection es données personelles

### 1. Principe fondamentaux

- Application du RGPD depuis 2018 dans l'UE
- Propriété des données par les individus
- Responsabilité des organisations

#### Acteurs Clés

- Data Subject (individus)
- Data Controller (Organisme du contrôle)
- Data Processor (Sous-traitant)

### Droits Obligatoire

#### Droit des individus

- Consultation des données
- Rectification
- Suppression
- Information en cas de violation

### Données Personelles

#### Définition

- Identifiant directs :
   - Nom
   - Adresse
   - Email
   - Numéro de téléphone
- Identifiant indirects 
- Attributs personnels (santé, religion, etc...)

### Gestion des données

#### Conversation 

- Durée limitée au traitement
- Effacement après utilisation
- Cas spéciaux (comptabilité, etc...)

### Territorialisation

#### Contraintes

- Hébergement territorial
- Protection des transferts
- Sécurisation du développement
- Gestion du Cloud
